### problems
* LaTex Error: File "fullpage.sty" not found.

you need `sudo apt-get install texlive-latex-extra` or `sudo apt-get install texlive-latex-extra --no-install-recommends`

* for chinese support 
 
 `sudo apt-get install latex-cjk-all` may help
